# UI Foundry Input Component Generator

Standalone visual text-input generator built with Next.js.

This app is a focused text-input studio for form fields, branded entry surfaces, validation-ready states, and styled input patterns that can be edited visually and exported as code.

## Product summary

This generator is designed for:

- designers building forms, auth screens, settings pages, and checkout flows
- developers who want editable input starting points without rebuilding every field from scratch
- freelancers and agencies preparing branded form systems for client work
- buyers who want a standalone text-input tool that fits the broader UI Foundry suite

## What the product includes

- section-based editing controls
- live preview panel
- code view with copy support
- file export support
- searchable preset library
- generated premium preset catalog with filters, paging, and surprise-me apply
- richer platform-attribute controls including `id`, `title`, `tabIndex`, `dir`, `lang`, `inputMode`, `enterKeyHint`, `autoCapitalize`, `autoCorrect`, and numeric `min` / `max` / `step`
- deeper field editing with dedicated field-attribute and adornment/action sections
- preview/export-safe prefix and suffix content, clear actions, password reveal, and structured supporting-message composition
- undo and redo controls
- self-contained standalone app structure

## Editing coverage

The editor currently includes these control areas:

- Presets
- Basics
- Field
- Styling
- Typography
- States
- Effects
- Labels
- Adorn
- A11y

These sections cover field setup, platform attributes, adornments and actions, styling, typography, states, decorative effects, label and supporting-message behavior, and a broader HTML/accessibility attribute surface for production-ready text fields.

## Export workflow

The input preview and code output are driven by the same current field state so users can design visually and export the matching implementation.

## Why this product is useful

This is more useful than a fixed input snippet because it gives the buyer a real field-building studio for reusable form directions.
